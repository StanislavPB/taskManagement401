
Пользователь (User)
ID: Уникальный идентификатор пользователя.
Имя (Name): Имя пользователя.
Пароль (Password): Хешированный пароль пользователя.
Логин (Login): Уникальный логин пользователя.
Статус (Status): Статус пользователя (босс или сотрудник).
Список проектов (Projects): Проекты, в которых участвует пользователь.

Проект (Project)
ID: Уникальный идентификатор проекта.
Название (Name): Название проекта.
Описание (Description): Описание проекта.
Статус (Status): Статус выполнения проекта (boolean).
Коллекция пользователей (Users): Пользователи, работающие над проектом.
Коллекция задач (Tasks): Задачи, связанные с проектом.

Задача (Task)
ID: Уникальный идентификатор задачи.
Название (Name): Название задачи.
Дата окончания (EndDate): Дата окончания задачи (LocalDate).
Список пользователей (Users): Пользователи, отвечающие за выполнение задачи.
Приоритетность (Priority): Приоритетность задачи (высокий, низкий, средний).
Статус выполнения (Status): Статус выполнения задачи (boolean).
Список комментариев (Comments): Комментарии к задаче.

Комментарий (Comment)
ID: Уникальный идентификатор комментария.
Пользователь (User): Пользователь, оставивший комментарий.
Сообщение (Message): Текст комментария.

Сообщение (Message)
ID: Уникальный идентификатор сообщения.
ID-from: ID пользователя-отправителя.
ID-to: ID пользователя-получателя.
Сообщение (Message): Текст сообщения.
LocalDate: Дата отправки сообщения.
