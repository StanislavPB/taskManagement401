
Документация и Описание пункта меню Босса "Check progress of project":

Класс CheckProjectMenu предназначен для отображения списка активных проектов и позволяет пользователю,
в данном случае боссу, отмечать проекты как завершенные.
Для этого используются два сервиса: ProjectStatusCheckService для проверки наличия незавершенных задач в проекте
и ProjectStatusChangeService для изменения статуса проекта на "завершенный".

Процесс работы метода checkAndChangeProjectStatus() выглядит следующим образом:

1. Выводит список всех активных проектов, используя метод findAllActiveProjects() из ProjectRepository.

2. Для каждого проекта отображает его идентификатор, название и задачи,
включая сроки выполнения и статус каждой задачи. Если срок выполнения задачи уже истек,
информация выводится красным цветом, а если задача еще активна - зеленым.

3. Запрашивает у пользователя идентификатор проекта, который он хочет отметить как завершенный.

4. Использует ProjectStatusCheckService для проверки, есть ли в выбранном проекте незавершенные задачи.

5. Если в проекте есть незавершенные задачи, запрашивает подтверждение у пользователя
на изменение статуса проекта, предупреждая о наличии таких задач.

6. При получении подтверждения от пользователя, ProjectStatusChangeService изменяет статус проекта
на "завершенный" и сообщает об этом пользователю.

Этот процесс помогает обеспечить тщательное управление проектами,
позволяя боссу иметь полный контроль над статусом проектов и задач внутри них,
и обеспечивая, что проекты не будут отмечены как завершенные, пока в них есть незавершенные задачи.
