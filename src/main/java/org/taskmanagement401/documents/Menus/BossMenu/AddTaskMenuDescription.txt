
Описание выполнения команды 3 - "AddTaskMenu" в BossMenu:

Класс AddTaskMenu отвечает за добавление новых задач в проекты в системе управления проектами.
Он предоставляет интерфейс боссу для выбора проекта и добавления в него задач.

Основные компоненты:
projectRepository: Репозиторий проектов для доступа к данным проектов.
taskRepository: Репозиторий задач для доступа и управления задачами.

Процесс работы:

Вывод списка активных проектов:
При активации меню выводится список всех активных проектов с их ID и названиями.
Пользователю предлагается выбрать проект для добавления задачи, вводя его ID.

Выбор проекта:
Пользователь вводит ID проекта, к которому хочет добавить задачу.
Если введенный ID корректен, выводится информация о существующих задачах в проекте.
Если проект не содержит задач, выводится соответствующее сообщение.

Добавление задачи:
Пользователю предлагается ввести параметры новой задачи, включая название, дату завершения и приоритет.
После получения параметров задачи создается объект TaskDto и передается в сервис TaskAddService
для регистрации задачи в системе.

Регистрация задачи:
Сервис TaskAddService обрабатывает данные задачи, проверяет их корректность и добавляет задачу в репозиторий задач.
В случае успешного добавления выводится сообщение об успешной регистрации задачи.
Если произошла ошибка, выводится сообщение об ошибке.


- Взаимодействие с пользователем осуществляется через консоль, где четко описывается каждый шаг
  и требуется ввод данных от пользователя.
- Использование сервисов для добавления задач обеспечивает разделение логики
  и упрощает тестирование и поддержку кода.
- Проверка наличия введенного проекта по ID и обработка возможных ошибок ввода повышает устойчивость
  приложения к некорректным действиям пользователя.
