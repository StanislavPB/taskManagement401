
Описание выполнения команды 8 - "CheckCommentMenu" в BossMenu:

Класс CheckCommentMenu предназначен для просмотра комментариев к задачам в рамках выбранного проекта.
Это позволяет боссу или менеджеру проекта следить за обсуждением задач и оперативно реагировать
на комментарии участников проекта.

Процесс работы с комментариями:

Выбор проекта для просмотра комментариев:
На первом этапе боссу предлагается выбрать проект из списка активных проектов,
чтобы проверить комментарии к его задачам.

Проверка на валидность выбора:
Если выбор оказывается невалидным (например, выбранный проект не существует), выводится сообщение об ошибке.
В противном случае, процесс продолжается.

Просмотр комментариев:
С использованием сервиса CheckTaskCommentService извлекается информация о комментариях к задачам в выбранном проекте.
Комментарии группируются по задачам.

Вывод комментариев:
Для каждой задачи в проекте выводится список комментариев.
Каждый комментарий включает имя пользователя, оставившего комментарий, и текст сообщения.



- Класс CheckCommentMenu упрощает процесс мониторинга общения внутри команды проекта,
  предоставляя централизованный доступ ко всем комментариям задач.
- Эффективная организация вывода комментариев по задачам позволяет быстро ориентироваться в текущем обсуждении
  и при необходимости вносить свои комментарии или задавать вопросы.
- Валидация выбора проекта и наличие комментариев обеспечивают корректное отображение информации
  и избегание ошибок в процессе работы.
