
Описание выполнения команды 7 - "ChatMenu" в BossMenu:

Класс ChatMenu обеспечивает взаимодействие с системой чата в приложении,
позволяя пользователю выбирать собеседника для обмена сообщениями и отправлять новые комментарии.

Процесс работы с чатом:

Выбор собеседника:
На начальном этапе пользователю предлагается список всех доступных контактов для чата.
Этот список формируется на основе данных из UserRepository, исключая текущего пользователя из списка контактов.

Проверка на валидность выбора:
Если выбор пользователя невалиден (например, выбран несуществующий контакт), выводится сообщение об ошибке.
В противном случае процесс продолжается.

Отображение истории сообщений:
После успешного выбора собеседника отображается история сообщений с этим контактом.
Для получения и отображения сообщений используется сервис ChatService.

Отправка нового сообщения:
Пользователю предлагается ввести текст нового сообщения.
Если сообщение не пустое, оно отправляется выбранному контакту через ChatService,
и пользователю выводится ответ о результате отправки.

Завершение работы с чатом:
Если пользователь вводит пустую строку вместо сообщения, это интерпретируется как желание выйти из режима чата.


- Класс ChatMenu облегчает общение между пользователями в системе, предоставляя простой интерфейс для отправки
  и просмотра сообщений.
- Валидация на каждом этапе обеспечивает корректность ввода данных и помогает избегать ошибок в процессе общения.
- Использование сервисов для взаимодействия с данными чата позволяет упростить архитектуру приложения
  и облегчить его тестирование и поддержку.
