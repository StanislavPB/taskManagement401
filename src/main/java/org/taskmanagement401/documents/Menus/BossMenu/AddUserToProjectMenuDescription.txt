
Описание выполнения команды 4 - "AddProjectMenu" в BossMenu:

Класс AddUserToProjectMenu управляет процессом добавления пользователей к проекту.
Он позволяет боссу выбрать проект и пользователя для добавления в этот проект.

Ключевые компоненты:

projectRepository: Репозиторий проектов для доступа к данным проектов.
userRepository: Репозиторий пользователей для доступа к данным пользователей.

Процесс работы:

Выбор проекта:
Сначала боссу предлагается список активных проектов для выбора. Босс выбирает проект по его ID.
Если выбор некорректен (например, выбран несуществующий проект), выводится сообщение об ошибке.

Выбор пользователя для добавления:
После успешного выбора проекта выводится список пользователей, которые могут быть добавлены в этот проект.
Босс выбирает пользователя по его ID. Если все пользователи уже добавлены в проект,
выводится соответствующее сообщение.

Добавление пользователя в проект:
После выбора пользователя система пытается добавить его в выбранный проект.
Процесс добавления управляется сервисом AddUserToProjectService,
который проверяет возможность добавления и выполняет операцию.

Ответ системы:
После попытки добавления пользователя в проект система выводит результат операции.
В случае успеха выводится сообщение об успешном добавлении пользователя.
В противном случае выводится сообщение об ошибке.


- Предоставление выбора из активных проектов и доступных пользователей упрощает процесс добавления
  и предотвращает ошибки.
- Валидация выбора проекта и пользователя обеспечивает корректность данных перед добавлением.
- Использование сервисов для выполнения операции добавления облегчает тестирование и поддержание кода,
  а также позволяет переиспользовать логику в других частях системы.
