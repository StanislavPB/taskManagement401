
Описание выполнения команды 6 - "CheckProjectMenu" в BossMenu:

Класс CheckProjectMenu предназначен для реализации функциональности,
позволяющей менеджеру проекта (боссу) проверять статус задач в проектах
и при необходимости изменять статус проекта на "завершено".
Данный класс является частью пользовательского интерфейса системы управления проектами.

Основные Компоненты:

projectRepository: Ссылка на репозиторий проектов, позволяющая получить доступ к данным проектов.
checkService: Сервис для проверки наличия незавершенных задач в проекте.
changeService: Сервис для изменения статуса проекта на "завершено".

Процесс работы:

Выбор проекта:
В начале работы метод выводит список всех активных проектов с их ID и названиями.
Пользователю предлагается выбрать проект для изменения статуса, вводя его ID.

Проверка и вывод задач:
Для выбранного проекта выводится список задач, включая их ID, название, дату завершения и статус.
Задачи, срок выполнения которых истек, выводятся красным цветом, а еще активные - зеленым.

Подтверждение изменения статуса:
Если в проекте есть незавершенные задачи,
система запрашивает подтверждение на изменение статуса проекта на "завершено",
учитывая наличие невыполненных задач. Если пользователь соглашается, процесс продолжается.

Изменение статуса проекта:
После получения подтверждения, сервис changeService изменяет статус проекта на "завершено".
Пользователю выводится соответствующее сообщение об успешном изменении статуса или о том,
что проект с указанным ID не найден.

Важные моменты:
- Все действия сопровождаются выводом информационных сообщений для пользователя,
  что делает процесс взаимодействия с системой интуитивно понятным.
- Использование цветового выделения для задач помогает быстро ориентироваться в статусах задач по проектам.
- Класс взаимодействует с репозиторием и сервисами для выполнения своих функций,
  что обеспечивает гибкость и расширяемость системы управления проектами.
