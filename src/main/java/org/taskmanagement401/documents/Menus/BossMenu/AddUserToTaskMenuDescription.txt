
Описание выполнения команды 5 - "AddUserToTaskMenu" в BossMenu:

Класс AddUserToTaskMenu предназначен для добавления пользователя к задаче в рамках проекта.
Он позволяет боссу управлять распределением задач среди сотрудников,
выбирая специфические задачи и назначая их конкретным пользователям.

Основной процесс:

Выбор проекта:
Выводится список всех активных проектов.
Босс выбирает проект, в котором нужно назначить задачу пользователю, по ID проекта.

Выбор задачи:
После выбора проекта выводится список всех задач, связанных с этим проектом.
Босс выбирает конкретную задачу для назначения пользователю, используя ID задачи.

Выбор пользователя:
Далее, предлагается список всех пользователей, которые могут быть назначены на выбранную задачу.
Босс выбирает пользователя по ID.

Назначение пользователя на задачу:
Выбранный пользователь назначается на выбранную задачу.
Процесс назначения управляется сервисом AddUserToTaskService.

Отображение результата:
По завершении процесса назначения на экран выводится сообщение с результатом.
В случае успешного добавления пользователя к задаче, выводится подтверждение,
в противном случае - сообщение об ошибке.


- Класс обеспечивает удобный интерфейс для босса по управлению распределением задач среди сотрудников
  в рамках выбранных проектов.
- Валидация выбора на каждом этапе (проект, задача, пользователь) предотвращает ошибочное назначение
  и обеспечивает корректность процесса.
- Использование сервисов для операций добавления пользователя к задаче упрощает тестирование и поддержку кода.
