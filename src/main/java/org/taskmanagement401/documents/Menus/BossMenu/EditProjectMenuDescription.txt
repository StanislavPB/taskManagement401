
Описание выполнения команды 2 - "Editing project" в BossMenu:

Класс EditProjectMenu служит для реализации функционала редактирования описания проекта
в системе управления проектами, доступного боссу.
Он предоставляет интерфейс для выбора проекта из списка всех проектов и последующего изменения его описания.

Процедура Редактирования Описания Проекта:

Выбор Проекта для Редактирования:
Сначала выводится приглашение "Select a project to edit:",
после чего перечисляются все доступные проекты с их ID,
используя метод UserTalkService.printAllProjects(projects).
Пользователю предлагается ввести ID проекта, который он хочет редактировать,
через метод UserInput.inputPositiveInt("Choose a project by ID to edit: ").

Проверка и Выбор Проекта:
Из списка проектов projects выбирается проект с введенным ID.
Если такой проект существует (optionalProject.isPresent()), процесс продолжается.
В противном случае выводится сообщение "Invalid project selection." и процесс редактирования завершается.

Ввод Нового Описания:
Для выбранного проекта выводится его текущее описание ("Current description: "),
после чего пользователю предлагается ввести новое описание через UserInput.inputText("Enter new description: ").

Сохранение Изменений:
С использованием сервиса ProjectEditService, вызывается метод editProjectDescription(selectedProject.getId(), newDescription),
который обновляет описание проекта на введенное пользователем.
После обновления выводится ответ сервиса, содержащий результат операции.

Важные Моменты:
- Валидация ввода пользователя осуществляется на нескольких этапах,
  включая проверку наличия проекта с указанным ID и корректность введенного нового описания.
- В случае успеха, пользователю предоставляется обратная связь о том, что описание проекта успешно обновлено.
- В случае возникновения ошибок (например, если проект не найден или введены некорректные данные),
  пользователю выводятся соответствующие сообщения об ошибках.
