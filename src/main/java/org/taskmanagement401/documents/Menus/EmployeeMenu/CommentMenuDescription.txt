
Описание выполнения команды - "CommentMenu" в EmployeeMenu:

Класс CommentMenu позволяет сотрудникам проекта добавлять комментарии к задачам, в которых они участвуют.
Это упрощает обмен информацией и ускоряет процесс работы над проектом,
обеспечивая более эффективную коммуникацию в команде.

Процесс добавления комментария к задаче:

Проверка на наличие активных задач у пользователя:
Сначала система проверяет, есть ли у пользователя задачи для выполнения. Если задач нет, выводится соответствующее сообщение.

Выбор задачи для комментирования:
Если у пользователя есть задачи, ему предлагается выбрать одну из них для добавления комментария. Выводится список задач, и пользователь вводит номер задачи, к которой хочет добавить комментарий.

Ввод текста комментария:
После выбора задачи пользователь вводит текст комментария.

Обработка и добавление комментария:
Введенный комментарий обрабатывается сервисом CommentService, который добавляет комментарий к выбранной задаче в репозиторий комментариев.

Отображение результата:
По завершении добавления комментария выводится ответ системы о статусе операции. Если комментарий добавлен успешно, выводится сообщение "Status ok".



- Класс CommentMenu обеспечивает важный аспект взаимодействия в команде,
  позволяя участникам проекта активно обсуждать детали выполнения задач и делиться мнениями.
- Простота добавления комментариев способствует более эффективной коммуникации между сотрудниками,
  что важно для совместной работы над проектом.
- Проверка на наличие активных задач у пользователя и валидация вводимых данных помогают избежать ошибок
  и недопонимания в процессе общения.
