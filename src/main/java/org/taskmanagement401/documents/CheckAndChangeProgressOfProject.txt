
Документация и Описание пункта меню Босса "Check and change progress of project":

Назначение:
Класс CheckProjectMenu разработан для использования в системе управления проектами и задачами,
обеспечивая функционал для руководителя проекта (босса) по изменению статуса проекта на "выполнено".
Он позволяет выбирать проект из списка активных проектов и изменять его статус после удостоверения в том,
что все связанные с проектом задачи выполнены.

Основной функционал:

Вывод списка активных проектов:
При активации, боссу представляется список всех активных проектов, включая детали такие как ID проекта,
его имя, а также список задач с указанием их статуса выполнения, даты окончания,
и выделение просроченных задач красным цветом для лучшей наглядности.

Выбор проекта для изменения статуса:
Руководитель проекта может выбрать конкретный проект по его ID для изменения статуса проекта на "завершенный".

Проверка наличия невыполненных задач:
Перед изменением статуса проекта система автоматически проверяет,
остались ли в выбранном проекте невыполненные задачи.
Если обнаруживаются невыполненные задачи, боссу предлагается сделать осознанный выбор об изменении статуса,
подтверждая готовность закрыть проект несмотря на наличие незавершенных работ.

Изменение статуса проекта:
После получения подтверждения от босса, статус проекта обновляется до "выполнено",
и пользователю выводится сообщение об успешном изменении статуса.


При обнаружении невыполненных задач, особенно просроченных, текст задач выводится красным цветом,
в то время как активные задачи, сроки которых еще не наступили, выводятся зеленым цветом.
Это позволяет боссу лучше визуализировать текущее состояние проекта и принимать обоснованные решения.

Для изменения цвета текста в консоли используются ANSI escape коды.

Эта функциональность критически важна для поддержания актуального состояния проектов и задач,
обеспечивая эффективное управление ресурсами и соблюдение сроков.
