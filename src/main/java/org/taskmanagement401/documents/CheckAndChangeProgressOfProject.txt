
Документация и Описание пункта меню Босса "Check and change progress of project":

Назначение:
Класс CheckProjectMenu предназначен для работы босса с системой управления задачами в рамках проектов.
Он позволяет боссу выбрать проект из списка активных проектов и изменить его статус на "выполнено",
при условии, что все задачи в проекте выполнены.

Основной Функционал:
Вывод списка активных проектов:
При активации меню боссу предлагается список всех активных проектов с указанием их ID, имени и списка задач,
включая информацию о статусе выполнения каждой задачи.

Выбор проекта для изменения статуса:
Босс может выбрать проект по его ID с целью изменения его статуса на "выполнено".

Проверка наличия невыполненных задач:
Прежде чем изменить статус проекта, система проверяет наличие невыполненных задач в выбранном проекте.
Если такие задачи есть, боссу предлагается подтвердить изменение статуса проекта,
учитывая наличие незавершенных работ.

Изменение статуса проекта:
По завершении проверки и получении подтверждения от босса, статус проекта изменяется на "выполнено",
и об этом выводится соответствующее сообщение.
