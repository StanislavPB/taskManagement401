
Описание выполнения команды 2 - "Editing project" в BossMenu:

1. Выбор проекта:
Сначала пользователю предлагается выбрать проект из списка для его последующего редактирования.
Список проектов получается из репозитория,
и каждому проекту присваивается порядковый номер для удобства выбора.

2. Ввод номера проекта:
Пользователь вводит номер выбранного проекта через командную строку.

3. Проверка валидности выбора:
Код проверяет, что введенный номер находится в допустимом диапазоне.
Если выбор невалиден, выводится сообщение об ошибке, и процедура завершается.

4. Редактирование описания:
Если выбор валиден, пользователю предлагается ввести новое описание для выбранного проекта.

5. Применение изменений:
Создается экземпляр сервиса ProjectEditService для редактирования проекта.
Этот сервис использует метод editProjectDescription,
который изменяет описание проекта с учетом валидации.
В случае успеха или неудачи, сервис возвращает ответ с соответствующим статусом.

6. Вывод результата:
В конце пользователю выводится сообщение об успешности или неудаче операции редактирования.
